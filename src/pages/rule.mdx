# Potato 服务器游玩规则 第四版

《Potato 服务器游玩规则》（以下简称“规则”）是由您与 Curiousers! Studio（以下简称“CSUB”） 共同缔结并遵守的规则，请您在确认同意本规则之前或在游玩之前仔细阅读本规则。

除非您已阅读并接受本规则所有条款，否则您无权使用 Potato 服务。请确保您已经充分了解并同意该条款内容，并为自己的游戏账号在使用 Potato 服务时（包括游戏和社区内）的一切行为负责。

## 一、须知 {#1}

1. 如果需要加入 Potato 服务器，您需要享有拥有来自 Mojang 官方的 Minecraft 客户端。
2. 您在使用 Potato 服务中的测试服或正式服时可能无法继承所有游戏数据。一旦该周目（主题）测试服体验结束后（或您测试服账号已被移出白名单），您在该服务器下所有玩家数据（游戏内的等级、物品、建筑等）*可能*都将会被重置。因此造成的任何损失，均由您自行承担。（见`一:4`：我们通常会归档这些数据并提供下载。）
3. 我们每年会开放两到三个存档，根据时期分为：

    - 长期服（视情况而定）：分为两个阶段，开放时间约在每年的上半年或下半年（具体取决于寒假服和暑假服的开放时间）。
    - 暑假服：每年暑假期间。
    - 寒假服：每年寒假期间，根据农历新年日期调整。

    对于原版生存玩法的存档，CSUB 可能会提供基岩版支持。

4. CSUB 不会*未到更换存档的时期*而重置进度，包括世界存档和玩家数据信息。但为了高效利用服务器资源，如果您超过6个月未使用 Potato 服务， CSUB 有权在不提前通知您的情况下，对该账号及其账号下的游戏数据及相关信息采取删除等处置措施。此规则有以下几种例外情况：
    - 游玩时长大于 30 小时的玩家不会被清理。
    - 开发者、捐赠者、管理组成员玩家不会被清理。
    - 向 CSUB 提交（并获批准）账户数据保留申请的玩家不会被清理。

    除了以上这些例外情况，所有游戏内数据也不能在新的存档继承。一旦我们更换存档，旧的存档将被归档，并且提供下载。

## 二、游戏规则 {#2}

1. 尊重玩家和工作人员
    - 保护个人隐私信息
    - 尊重其他玩家的空间和财产
    - 保持友好
    - 不要未经授权使用他人的游戏账号或虚拟财产
    - 遵守我们的[社区](#community)规则
2. 管理组、维护组内所有人员不得利用职业便利影响玩家正常游戏，包括但不限于非法给与玩家物品、恶意攻击或封禁他人。玩家可根据本规则 [第四条](#4)第3节 向 CSUB 提交举报。
3. 不得进行以下干扰服务器的行为：
    - 以任何方式破坏、骇入、攻击 CSUB 服务器群组或非法影响服务器的正常进行
    - 恶意实施可能导致游戏服务器宕机、卡顿，或导致其他游戏玩家断连、延迟等情形使得玩家无法正常游戏的行为等
    - 对游戏服务器节点或玩家进行网络流量攻击等
4. 不可在游戏内：
    - 恶意攻击（言论或物理）其他玩家
    - 利用游戏、服务端的软件恶性漏洞实施获益，或给其他玩家造成不良影响
    - 冒充、欺诈，或设置陷阱坑害、困住其他玩家
    - 建造大型红石机器而不设全局开关
    - 建造不合适或令人反感的建筑
    - 向其他玩家提供不公平的好处及优势
    - 使用不合适的皮肤或 ID
    - 恶意盗窃、破坏、占有其他玩家或公有建筑、财产等
5. 禁止使用具有不公平优势的外挂或模组，包括但不限于：
    - X 光透视
    - 连点器、宏等
    - 建筑放置作弊
    - 非法移动
    - 修改玩家状态以达到作弊效果的
6. 我们允许使用这类模组：
    - 功能增强（BetterPVP、HUD+、小地图、血量显示等）
    - 性能优化（OptiFine、Iris 等）
    - 提供便利（蓝图、JEI、合成表等）
    - 其他能公平地增强游戏体验的 Mod。如果您不确定，可以在使用之前询问管理组。

## 三、社区规则 {#3}

1. 您在游戏内或社区内可发送文字，超文本链接，媒体文件及其他社交内容，包括但不限于群聊聊天、角色命名/昵称、游戏内建筑和媒介，但应当以尊重他人为前提。您发布、使用或传播的内容**不得**含有以下特征：
    - 人身攻击，无端滋事，不当暗示、影射等
    - 骚扰，寻衅，谩骂，诽谤，阴阳怪气等
    - 政治相关、不文明词汇
    - 血腥暴力，恐吓
    - 侵犯他人隐私
    - 恶意刷屏（但复读除外）
    - 与服务器内玩法无关的广告，推销等
    - 暗示或伪造身份以企图得到不正当利益的
    - 内容包含有争议的政治或公共事件
    - 宣传其他服务器、群聊等
    - 其他引起玩家不适的言论和话题

2. 在收到 `自称是 Potato 管理组的人员` 的**主动联系**时，您应当确保对方位于我们目前的[管理组列表](/community#admin)中。

## 四、举报与处罚 {#4}

1. 任何违反以上规则的行为都可能导致处罚，处罚内容视情节轻重，最高可达被永久封禁。
2. 如果您要举报某玩家的违规行为，请联系我们的 [管理组](/community#admin)。
3. 如果您要举报管理组或其某个成员的违规行为，请向 yukonisen 发起私信，或发送包含合理理由的电子邮件到 `yukonisen(at)curiousers.org`（请自行替换`(at)`为`@`）。CSUB 保证您的举报是匿名的。
4. 处罚或封禁时间由管理层讨论决定，通常情况不接受减免，延缓等要求，并保留追加处罚的权利。
5. 玩家被处罚后如有任何异议、意见等可继续联系管理组进行讨论。
6. 处罚细节是否需要被公开由管理层决定。

## 五、反作弊 {#5}

1. 为了保证玩家的游戏体验，PotatoEye（`PotatoEye Anti Cheat`，PAC） 会自动检测游戏内玩家的操作，并试图阻止任何异常的游戏行为，在必要时采取封禁措施。
2. PotatoEye 只会在检测到超出常理的行为（普通玩家根本无法完成的操作）时向管理组发出警告或采取行动，任何人无法干预。
3. 我们不保证 PotatoEye 的判断完全正确。如果您被误封禁，在管理员确认系误封后，将根据情况减免或免除处罚。
4. PotatoEye 的全部检测在服务端处理，并且不会收集与游戏内玩家无关的数据或个人信息。

## 六、补充 {#6}

1. Potato 有权在必要时变更本规则，请于相关页面查阅最新版本的规则条款。本规则变更后，如果您继续使用 Potato 服务，即视为您已接受变更后的规则。
2. 在您使用 Potato 服务时，我们有权收集您在 Potato 社区、官方群聊、游戏服务器内的游戏数据和行为等信息，用于包括但不限于改善优化游戏体验，收集违规证据等用途。管理组不会在调查事件之外调取这些信息。
3. 本规则的最终归属权属于 CSUB。
