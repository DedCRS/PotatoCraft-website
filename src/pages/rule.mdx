# Potato 游玩规则

## 一、规则 {#2}

作为 Potato 的一员，您必须遵守这些规则。

管理团队的每位成员都值得信赖，可以根据自己的意愿做出判断和调节行动，但如果您需要任何澄清或有任何疑虑，您可以直接联系管理团队中的任何人。

1. 尊重其他玩家的空间和财产，不要未经授权使用他人的游戏账号或虚拟财产
2. 不得以任何形式恶意干扰服务器，或以任何方式破坏、骇入、攻击 Potato 服务器群组或非法影响服务器的正常运行
3. 不得实施可能导致游戏服务器宕机、卡顿，或导致其他游戏玩家断连、延迟等情形使得玩家无法正常游戏的行为等
4. 除了要遵守社区规则，您也不能
    - 利用游戏、服务端的软件恶性漏洞实施获益
    - 建造红石机器而不设全局开关
    - 建造不合适或令人反感的建筑
    - 向其他玩家提供不公平的好处及优势
5. 禁止使用具有不公平优势的外挂或模组，包括但不限于：X 光透视、建筑放置、非法移动，或其他修改玩家状态以达到作弊效果的
6. 例外：我们允许使用公平模组：
    - 功能增强（BetterPVP、HUD+、小地图、血量显示等）
    - 性能优化（OptiFine、Iris 等）
    - 提供便利（蓝图、JEI、合成表等）
    - 其他能公平地增强游戏体验的 Mod。如果您不确定，可以在使用之前询问管理组。

## 三、社区规则 {#3}

Potato 及其社区允许任何人加入。在发布内容时，应当以尊重他人为前提。发布的内容不得包含：
    - 骚扰，寻衅，谩骂，诽谤，阴阳怪气等
    - **政治相关**、不文明词汇
    - 血腥暴力，恐吓
    - 保护自己和他人的隐私。避免分享任何人的个人信息。
    - 恶意刷屏（但复读除外）
    - 广告，推销等
    - 暗示或伪造身份以企图得到不正当利益的
    - 偏见和仇恨言论
    - **宣传其他服务器、群聊等**
    - 其他引起玩家不适的言论和话题

## 四、举报与处罚 {#4}

1. 任何违反以上规则的行为都可能导致处罚，处罚内容视情节轻重，最高可达被永久封禁。
2. 如果您要举报某玩家的违规行为，请联系我们的 [管理组](/community#admin)。
3. 如果您要举报管理组或其某个成员的违规行为，请向 yukonisen 发起私信，或发送包含合理理由的电子邮件到 `yukonisen(at)curiousers.org`（请自行替换`(at)`为`@`。Potato 保证您的举报是匿名的。
4. 处罚或封禁时间由管理层讨论决定，通常情况不接受减免，延缓等要求，并保留追加处罚的权利。
5. 玩家被处罚后如有任何异议、意见等可继续联系管理组进行讨论。
6. 处罚细节是否需要被公开由管理层决定。

## 五、管理理念 {#5}

作为一名管理团队成员，您被赋予了极大的信任和对服务器产生长远影响的能力，但需要知悉：

1. 管理的核心原则：
    - 公平与尊重：无论成员的年龄、种族、技能水平或身份如何，都要以公平和尊重对待
    - 透明度：对于管理员的行为原因应明确说明，并且服务器成员与管理员之间应开放讨论
    - 理解与信任：尊重成员之前的行为和在社区中的地位
2. 审核政策
    - 不得基于年龄、身份、技能水平或个人感受而对其他成员进行歧视。
    - 对于轻度违规，优先采取暂时封禁、踢出或禁言措施。
    - 对明显的偏见内容采取零容忍政策。
    - 尽可能以最不具破坏性的方式采取适度行动。
3. 处理：如果有必要封禁玩家，请务必提供详细的解释说明采取该行动的原因。不得容忍有害行为：要迅速处理仇恨和有害行为。

## 六、补充 {#6}

1. Potato 有权在必要时变更本规则。
2. 在您使用 Potato 服务时，我们有权收集您在 Potato 社区、官方群聊、游戏服务器内的游戏数据和行为等信息，用于包括但不限于改善优化游戏体验，收集违规证据等用途。管理组不会在调查事件之外调取这些信息。
3. 本规则的最终归属权属于土豆网络。
