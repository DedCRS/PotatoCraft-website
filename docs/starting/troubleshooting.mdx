# 故障排除

### 连接问题 {#connection}

1. 服务器积极拒绝连接
    - 检查 [服务器状态](https://stats.uptimerobot.com/305nWhXn1B)
    - 检查填写的地址是否正确
2. 服务器连接超时
    - 检查 [服务器状态](https://stats.uptimerobot.com/305nWhXn1B)
    - 检查填写的地址是否正确
    - 检查您的计算机网络畅通性
3. 终端（UPT）提示“连接至远程服务器失败”
    - 请联系管理员，这是一个故障
4. 终端提示“远程服务器拒绝加入”
    - 请联系管理员，这是一个故障
5. 终端提示“没有权限”
    - 可能出现在测试阶段，您的 ID　没有加入所需的权限
    - 如果您坚信这是一个服务器故障，请联系管理员
6. 连接已丢失
    - 检查您的网络稳定性
    - 如果您坚信这是一个服务器故障，请联系管理员
7. 终端提示“维护中”
    - 请等待维护结束，并且我们会根据维护时长对所有玩家进行补偿
8. 未知的主机名
    - 检查您的 DNS （域名服务器）设置
    - 请根据[本期服务器文档](/docs/intro)重新添加服务器地址
9. 终端提示“不明客户端”
    - 如已标注，请使用受支持的客户端（以及对应版本）连接

### 游戏体验 {#gameplay}

1. 很低的帧数
    - 检查您电脑 GPU 等设备的驱动软件更新
    - 检查您电脑上的设置项（省电模式、温度控制等）
    - ***笔记本请插上电源谢谢您***
    - 降低游戏画面（渲染区块、画面质量、光影效果等）
    - 再卡建议基岩版
2. 网络高延迟/丢包
    - 使用更稳定的网络连接方式，如有线连接
    - 使用游戏加速器，地区选择中国（或东亚）
    - 如果您在海外，请尝试加入我们位于日本的终端入口（若开放）
    - 对您的运营商（ISP）口诛笔伐
3. 无故断连
    - 使用更稳定的网络连接方式，如有线连接
    - 使用游戏加速器，地区选择中国（或东亚）
    - 如果您坚信这是一个服务器故障，请联系管理员
4. 我的电脑配置不够
    - 建议基岩版

### 反作弊 {#pac}

1. 我被报告错误的移动或放置
    - 检查安装的 mod
        - 已知一些控制器支持 mod 会导致该问题，如 MidnightControls 通过安装 [MidnightControlsExtra](https://www.curseforge.com/minecraft/mc-mods/midnightcontrols-extra) 解决
        - 一些不具有公平优势的 mod 可能导致该问题。请参考 [规则](/rule#2) 的第二条第五和第六小节，了解我们允许和禁止使用的 mod。
    - 我使用的是基岩版
        - 由于基岩版是互通转译，PotatoEye 不对由终端标记为基岩版的玩家进行检测。如果您使用基岩版正常游戏时被 PotatoEye 阻止，请联系管理员。
    - 也有可能是 PotatoEye 在发癫。
2. 我被 PotatoEye 封禁了
    - 请联系管理员。如果管理员最终认定您没有违规行为，您将被解除并获得对应补偿。
