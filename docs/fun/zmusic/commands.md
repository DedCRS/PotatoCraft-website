# 命令

## 玩家命令

| 命令格式                 | 示例                        | 说明               |
| ------------------------ | --------------------------- | ------------------ |
| /zm play <平台> <歌名>   | /zm play netease 错位时空   | 播放一个音乐       |
| /zm music <平台> <歌名>  | /zm music netease 错位时空  | 全服点播一个音乐   |
| /zm search <平台> <歌名> | /zm search netease 错位时空 | 通过关键词搜索音乐 |
